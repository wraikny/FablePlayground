# FablePlayground

see [FablePlayground](https://wraikny.github.io/FablePlayground).

## Setup

.NET, npm

```sh
$ npm install
$ dotnet tool restore
```

## Format

```sh
$ dotnet fake build -t format
```

## Dev

```sh
$ npm run start
```

## Build

```sh
$ npm run build
```
